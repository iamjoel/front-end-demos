果冻效果

果冻特效源自此网站 点这里，但是官方示例会改变初始的方块体积，设值需要技巧！

果冻效果分为长宽两个尺寸的变化，0.8*1.25=1，可以自行找到规律。下面是设值示例：

scale #1

  • type: scale
  • width: from 0.8 to 1.25
  • height: from 1.25 to 0.8
  • easing: bounce

scale #2

  • type: scale
  • width: from 1.25 to 0.8
  • height: from 0.8 to 1.25
  • easing: bounce
